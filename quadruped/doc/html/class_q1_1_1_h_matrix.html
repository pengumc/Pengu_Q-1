Detailed Description

Holds a 4x4 Homogeneous matrix.

indices of array_
00, 01, 02, 03,
04, 05, 06, 07,
08, 09, 10, 11,
12, 13, 14, 15

Row 4 is always assumed 0, 0, 0, 1
The red entries are referred to as the rotation matrix R
The blue entries are referred to as translation vecot T
