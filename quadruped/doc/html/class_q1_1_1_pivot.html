Detailed Description

represents a pivot of a leg

Pivots rotate around their Z-axis.
-abs_max_angle_ > + angle_ > + abs_max_angle_
offset_angle_ is meant to be the middle of the angle range. so the range is (offset - abs_max, offset + abs_max)

Pivots should be configured once, after that only angle should be changed
Usage:

  1. Create instance
    Pivot pivotA = Pivot(NULL); // no parent
    Pivot pivotB = Pivot(pivotA.framep()); \\ with pivotA as parent
  2. configure the orientation of the pivot.
    pivotA.ConfigureRot(X_AXIS, 1.57);
  3. set the abs_max_angle. It's been reset during set_axis and you can't do much until it's been set again
    pivotA.set_abs_max_angle(0.3); // 0.3 rad
  4. set the offset angle
    pivotA.set_offset_angle(1.57);
  • you can now change or set the angle all you want
    bool success = pivotA.set_angle(1.4);
  • you can retrieve the local frame H_frame_ or the frame relative to the origin H_rel_
    HMatrix local = pivotA.H_frame();
    HMatrix relative = pivotA.H_frame();
    // PivotA doesn't have a parent, no parent = origin
    // so relative is the same as local in this case

    we now have a pivot that's been rotated around the x-axis by a 90 degrees. it's actuated rotation is around its own z-axis and in neutral it makes an angle of 1.57 rad. it can have any angle between 1.27 and 1.87 rad
